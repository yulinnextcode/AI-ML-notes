<summary><h1>3. Network common commands</h1></summary>

- check ip address
```
ipconfig    #windows

ifconfig    #Linux
ip addr
```
- check ip connection
```
ping
```

- telnet command, check port connection
```
telnet 
```

- traceroute
```
tracepath      # in Linux

tracert.exe    # in Windows
```
